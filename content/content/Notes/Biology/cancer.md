Classes of genes
* Oncogene -- promotes growth
	* These for example, can be genes that encode for cyclins
		* Example: G1 cyclin can be upregulated through growth signals, wnt, cytokines, etc.
		* G1-cyclin CDK phosphorylates Rb, which causes it to leave E2F so that it can start transcribing the G1-S cyclin gene
		* ![[Screenshot 2026-01-08 at 4.45.50 PM.png]]
	* before mutation -- protooncogene
	* "stuck accelerator"
* Tumor suppressor 
	* inhibit growth
	* "defective breaks"
	* E.g. Rb (retinoblastoma)
* Caretakers
	* involved in genome repair / integrity


If you are heterozygous in an gene for a disease, than that means that you are *predisposed* to the disease. This is because you are only 1 mutation away from lacking the functional version of the gene. Having a mutation in the same gene in two different chromosomes is much less likely!

Things that cause loss of heterozygosity
1. point mutation
2. loss of expression
3. deletion of gene
4. Loss of a chromosome (when cell division is happening, you can accidentally pull both chromosomes to the same side of the metaphase plate)
